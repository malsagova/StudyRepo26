# Анализ данных диспансеризации: Machine Learning в медицине

## Описание проекта
Учебный проект для магистрантов 1 года медицинского университета по предмету "Наука о данных в медицине". 

Проект состоит из **3 последовательных семинаров**, каждый из которых решает конкретную клиническую задачу на основе данных диспансерных осмотров 1000 пациентов aged 40-65 лет.

## План семинаров

| Семинар | Тема | Домашнее задание | Методы |
|---------|------|-----------------|--------|
| **1** | Разведочный анализ и PCA | ДЗ 3 | PCA, визуализация |
| **2** | Прогнозирование ССЗ-риска | ДЗ 4 | SVM, классификация |
| **3** | Скрининг преддиабета | ДЗ 5 | Fuzzy kNN, нечёткая логика |
