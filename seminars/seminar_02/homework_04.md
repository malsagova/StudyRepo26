# Задание 4: SVM для прогнозирования ССЗ-риска

## Клиническая задача
Применить SVM алгоритм для прогнозирования сердечно-сосудистого риска среди пациентов 40-65 лет, прошедших диспансеризацию.

## Требования к выполнению

### Часть 1: Подготовка данных (20%)
- [ ] Выбрать признаки для модели (8 показателей)
- [ ] Заполнить пропуски средним
- [ ] Стандартизировать данные
- [ ] Разделить на train/test (70/30, стратификация)

### Часть 2: Обучение модели (30%)
- [ ] Обучить SVM с RBF ядром
- [ ] Использовать `class_weight='balanced'`
- [ ] Включить `probability=True` для прогноза вероятностей
- [ ] Получить предсказания на тестовой выборке

### Часть 3: Оценка качества (30%)
- [ ] Рассчитать метрики: Accuracy, Precision, Recall, F1
- [ ] Построить матрицу ошибок
- [ ] Построить ROC-кривую с AUC
- [ ] Проанализировать 5 порогов вероятности (0.3-0.7)

### Часть 4: Клиническая интерпретация (20%)
- [ ] Интерпретировать AUC ROC
- [ ] Объяснить чувствительность и специфичность
- [ ] Дать рекомендации по выбору порога для:
  - Массового скрининга
  - Рутинной практики
  - Подтверждения диагноза
- [ ] Привести пример прогноза для 1 пациента

## Критерии оценки

| Критерий | Баллы | Описание |
|----------|-------|----------|
| Модель обучена корректно | 25 | Без утечки данных |
| Метрики рассчитаны верно | 25 | Все метрики присутствуют |
| Визуализации (матрица, ROC) | 20 | Графики информативны |
| Клинические рекомендации | 30 | Практическая применимость |


## Подсказки
- Используйте `stratify=y` при разделении для сохранения баланса классов
- Для ROC-кривой: `roc_curve(y_test, y_proba)`
- Порог 0.3 — высокая чувствительность (скрининг)
- Порог 0.7 — высокая специфичность (подтверждение)
